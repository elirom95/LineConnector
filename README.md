Diese Anwendung stellt Linien graphisch dar.

Verbundene Linien werden in derselben Farbe dargestellt. Zwei Linien gelten als verbunden, wenn sie in einem Punkt übereinstimmen und in diesem Punkt keine andere Linien enden oder beginnen.

Als Datenquelle werden pro Linie vier Gleitkommazahlen entgegengenommen, die die x- und y-Koordinaten der beiden Endpunkte der jeweiligen Linie repräsentieren.

Gleichfarbige Linien werden als Linienzug bezeichnet. Die Linienzüge werden im Backend und in der Darstellung nach Länge geordnet ausgegeben.
